## Overview

This is the personal portfolio of **Kamogelo Legae**, an ICT student and full-stack developer.  
The portfolio highlights projects, skills, and provides a way to connect professionally. Built using **HTML, CSS, JavaScript, and Bootstrap 5**.

---

## Features

- **Responsive Design:** Optimized for mobile, tablet, and desktop.
- **Hero Section:** Animated typing effect showcasing key roles.
- **About Me Section:** Personal info, skills, and a downloadable CV.
- **Projects Section:** Interactive project cards with hover effects.
- **Contact Section:** Contact form integrated for email submissions.
- **Accessible & Semantic:** Proper alt attributes, labels, and semantic HTML elements.
- **Smooth Scrolling:** Seamless scrolling between sections.

---

## TechStack

- HTML5, CSS3, JavaScript
- Bootstrap 5
- Bootstrap Icons
- Hosted on Netlify
